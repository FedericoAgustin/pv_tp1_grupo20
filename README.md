# PROYECTOS EDUCATIVOS 2026

## Descripcion

Una pagina web que permite ver un listado de distintos proyectos educativos, detalles de un proyecto y la informacion del usuario.

## Tecnologias utilizadas

- HTML
- CSS
- Git
- GitHub
- Se utilizo validador de html para verificar la sintaxis correcta.

## Estructura De La Pagina

- index.html → Pagina Principal (Pagina de inicio con el nombre de la plataforma, una seccion de estadisticas relacionadas a proyectos totales, tareas realizadas en el dia y mensajes pendientes. Ademas un apartado de nomvedades donde se notifica las actividades realizadas por distintos usuarios.)
- proyectos.html → Listado de proyectos educativos (Capacitacion Docente, Cursos Online, Biblioteca Digital, Olimpiadas y Orientacion Vocacional. Se agrego un filtro para ordenar los proyectos por categoria como: En curso, finalizado, formacion, año.)
- detalle.html → Detalle de un proyecto (Se amplia la informacion sobre el curso de capacitacion docente en herramientas digitales, con enlaces a recursos como pdf, drives,  repositorios en github.)
- perfil.html → Informacion del usuario (Se detalla la informacion sobre el usuario, Datos personales como la institucion, cursos en los que participa, correo electronico, numero de telefono y demas datos personales.)
- styles.css → Paginas de Estilos (Se usaron variables globales, selectores de clase, universales, tambien grid para una correcta organizacion de la pagina, uso del hover para añadir interacciones.)
